            \begin{block}{Introduction \& Setup}

                The \textbf{topologist's sine curve} is a classical counterexample in topology illustrating that

                $$
                    \text{Connected} \;\not\Rightarrow\; \text{Path Connected}.
                $$

                Following the informal exposition in \textbf{K. Conrad's paper}, and under the supervision of \textbf{Prof. David Loeffler}, this project formalizes the result. The generalized theorem has now been successfully \textbf{merged}.

                We define the sets as subsets of $\mathbb{R}^2$:

                \begin{lstlisting}
def S : Set (ℝ × ℝ) := sine_curve '' pos_real
def Z : Set (ℝ × ℝ) := {(0,0)}
def T : Set (ℝ × ℝ) := S ∪ Z
\end{lstlisting}

            \end{block}

            \begin{block}{The Topologist's Sine Curve}

                \vspace{0.5cm}
                \begin{center}
                    \begin{tikzpicture}
                        \begin{axis}[sine axis]
                            \addplot [curveblue, thick, domain=0.1:1.15, samples=150] {sin(deg(1/x))};
                            \addplot [curveblue, thick, domain=0.005:0.1, samples=850] {sin(deg(1/x))};
                            % The Origin
                            \addplot [dotred, mark=*, mark size=2.5pt] coordinates {(0,0)};
                        \end{axis}
                    \end{tikzpicture}
                \end{center}
                \vspace{0.5cm}

                The curve $S = \{(x,\sin(1/x)) : x > 0\}$ oscillates infinitely as $x \to 0$. Adding the origin produces the space $T = S \cup \{(0,0)\}$.

            \end{block}

            \begin{block}{Part 1: $T$ is Connected}

                $S$ is the continuous image of $(0,\infty)$. This reduces to a single explicit proof term by chaining continuity lemmas with the reverse application operator:

                \begin{lstlisting}
lemma S_is_conn : IsConnected S :=
  isConnected_Ioi.image sine_curve <|
    continuous_id.continuousOn.prodMk <|
    continuous_sin.comp_continuousOn <| ...
\end{lstlisting}

                Since $S \subseteq T \subseteq \overline{S}$, $T$ is also connected.
                To prove $(0,0) \in \overline{S}$, we construct the sequence $f(n) = (\frac{1}{n\pi}, 0)$, which remains in $S$ because $\sin(n\pi) = 0$.

                Filters expresses this naturally:

                \begin{lstlisting}
let f : ℕ → ℝ × ℝ :=
  fun n => ((n * Real.pi)⁻¹, 0)

have hf : Tendsto f atTop (𝓝 (0,0)) := ...
have hf' : ∀ᶠ n in atTop, f n ∈ S := ...
\end{lstlisting}

            \end{block}

        \end{column}

            \begin{block}{Part 2: Path-Connectedness \& API Friction}

                To prove that $T$ is not path-connected, assume there exists a continuous path $p : [0,1] \to T$ connecting the origin $(0,0)$ to a point of $S$.

                Define $t_0$ as the last time the path lies on the $y$-axis:

                \begin{lstlisting}
let t₀ : unitInterval :=
  sSup {t | (p t).1 = 0}
\end{lstlisting}

                \textbf{Subtype Friction:}
                Working with paths requires evaluating over the \texttt{unitInterval} subtype.
                Transferring inequalities to the underlying \texttt{ℝ} necessitates frequent
                casting and lemmas like \texttt{Subtype.coe\_lt\_coe} to enable the \texttt{linarith} tactic.

                \vspace{0.5cm}
                Because the path is continuous at $t_0$, choosing $\varepsilon = 1/2$ bounds the path near $(0,0)$:

                \begin{lstlisting}
obtain ⟨δ, hδ, ht⟩ :
  ∃ δ > 0, ∀ t,
  dist t t₀ < δ →
  dist (p t) (p t₀) < 1/2
\end{lstlisting}

            \end{block}

            \begin{block}{Continuity Visualization}

                \vspace{0.5cm}
                \begin{center}
                    \begin{tikzpicture}
                        \begin{axis}[sine axis]
                            \addplot [curveblue, thick, domain=0.1:1.15, samples=150] {sin(deg(1/x))};
                            \addplot [curveblue, thick, domain=0.005:0.1, samples=850] {sin(deg(1/x))};

                            % Draws a perfect circle natively in the data coordinates

                        \end{axis}
                    \end{tikzpicture}
                \end{center}
                \vspace{0.5cm}

                Immediately after $t_0$, the path must enter $S$.

                Because the image of a connected set under the continuous $x$-coordinate map must be a connected subset of $\mathbb{R}$ (an interval), the path must cover an entire interval $[0,a]$ along the $x$-axis inside this small time window.

            \end{block}

        \end{column}

        \begin{column}{0.32\textwidth}

            \begin{block}{The Contradiction: Oscillation Peaks}

                To cover the interval $[0,a]$, the path must encounter the infinite oscillations of the sine curve.

                We isolate the sequence of points where $\sin(1/x) = 1$:

                \begin{lstlisting}
noncomputable def xs_pos_peak :=
  fun (k : ℕ) =>
    2 / ((4 * k + 1) * Real.pi)
\end{lstlisting}

                \vspace{0.5cm}
                \begin{center}
                    \begin{tikzpicture}
                        \begin{axis}[sine axis]
                            % Dashed line at y=1, expanded left and right to match new padding
                            \draw[gray, dashed, thick] (-0.55, 1) -- (1.15, 1);
                            \node[above, gray] at (-0.4, 1.0) {1};

                            \addplot [curveblue, thick, domain=0.1:1.15, samples=150] {sin(deg(1/x))};
                            \addplot [curveblue, thick, domain=0.005:0.1, samples=850] {sin(deg(1/x))};
                            \addplot [dotred, mark=*, mark size=2.5pt] coordinates {(0,0)};

                            % Sequence of peaks: x = 2/((4k+1)pi)
                            \addplot [dotred, mark=*, mark size=1.8pt, only marks] coordinates {
                                    (0.6366, 1) (0.1273, 1) (0.0707, 1) (0.0489, 1)
                                    (0.0374, 1) (0.0303, 1) (0.0254, 1) (0.0219, 1) (0.0192, 1)
                                };
                        \end{axis}
                    \end{tikzpicture}
                \end{center}
                \vspace{0.5cm}

                These peaks approach $x = 0$, so we extract a time $t' \in [t_0, t_0 + \delta)$ where the path evaluates to the peak:

                \begin{lstlisting}
obtain ⟨t', ht', hpath_t'⟩ : 
  ∃ t' ∈ Set.Icc t₀ t₁, (p t').2 = 1 := ...
\end{lstlisting}

                Because the norm dominates the absolute value of any component, we have:

                $$
                    \|p(t') - (0,0)\| \ge |p(t')_2| = 1 > 1/2,
                $$

                contradicting the continuity bound that strictly keeps the path inside the ball of radius $1/2$. Therefore $T$ cannot be path-connected.

            \end{block}

            \begin{block}{Conclusion}

                Combining both arguments yields the final witness:

                \begin{lstlisting}
theorem T_is_conn_not_pathconn :
  IsConnected T ∧ ¬IsPathConnected T :=
  ⟨T_is_conn, T_is_not_path_conn⟩
\end{lstlisting}

                A natural next step for future formalization is the \textbf{Warsaw circle}.
            \end{block}
